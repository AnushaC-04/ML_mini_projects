# Titanic Survival Prediction using Pipeline & ColumnTransformer

## 📌 Project Overview

This project predicts passenger survival on the Titanic dataset using Logistic Regression with an end-to-end Scikit-learn preprocessing workflow. The project demonstrates the use of Pipeline and ColumnTransformer to automate data preprocessing and model training.

## 🎯 Objectives

- Handle missing values using SimpleImputer
- Scale numerical features using StandardScaler
- Encode categorical features using OneHotEncoder
- Apply preprocessing using ColumnTransformer
- Build an end-to-end Machine Learning Pipeline
- Evaluate the model using classification metrics

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📂 Files

- `Titanic_pipeline.ipynb` – Complete notebook
- `Titanic-Dataset.csv` – Dataset

## 📈 Workflow

1. Data Exploration
2. Train-Test Split
3. Numerical & Categorical Feature Selection
4. Data Preprocessing with Pipeline
5. ColumnTransformer
6. Logistic Regression Model
7. Model Evaluation

## 🚀 Key Learning Outcomes

- Built separate preprocessing pipelines for numerical and categorical data.
- Automated preprocessing using Pipeline and ColumnTransformer.
- Prevented data leakage by fitting preprocessing only on the training set.
- Created a reusable end-to-end machine learning workflow.

## 📌 Conclusion

This project demonstrates industry-standard preprocessing techniques using Scikit-learn's Pipeline and ColumnTransformer, making the workflow efficient, reusable, and suitable for real-world machine learning applications.
